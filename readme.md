# Verificador de Palíndromos en Python

Este programa permite comprobar si una palabra o frase es un **palíndromo**, es decir, si se lee igual de izquierda a derecha que de derecha a izquierda (ignorando espacios y mayúsculas).

## 🚀 Cómo usarlo

1. Clona este repositorio o descarga el archivo `.py`.
2. Ejecuta el programa en tu terminal:
   ```bash
   python palindromo.py
